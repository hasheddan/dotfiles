# neovim

[neovim](https://neovim.io/) is a hyperextensible Vim-based text editor. The
majority of the `nvim` configuration in this repository is based on the
following references.

- [ThePrimeagen's `init.lua` repository](https://github.com/ThePrimeagen/init.lua)
- [Vineeth Voruganti setup guide](https://www.vineeth.io/posts/neovim-setup)
